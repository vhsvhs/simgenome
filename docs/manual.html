SimGenome

Simulated directed evolution of gene regulatory circuits

Written by Victor Hanson-Smith :: EMAIL victor.hanson-smith@ucsf.edu

Overview | Download and Install | Tutorial | Command Reference | Output Files

SimGenome is a software suite for simulating the evolution of a population of artificial genomes, each encoding a gene transcription regulatory circuit. This software is designed to test hypotheses about the evolution of regulatory circuits in response to a wide range of evolutionary and genomic conditions.

To get started, follow the links above to download and install, and then run SimGenome.

Overview

SimGenome implements a unified model of gene regulatory circuit evolution that includes multiple parameters to account for genome sequences, transcription factor DNA-binding preferences, strength of selection, and mutation rate. The unified model is illustred in the figure below, and described as follows:

(Fig. a) A population of individuals with unique genomes responds to a regulatory problem, defined as input stimuli and ideal output gene expression that would constitute maximum fitness. In this cartoon, each genome contains six genes, but in practical simulations, genomes contain hundreds or thousands of genomes. The thermodynamic model (T.M., Segal et al. 2008) translates each genome into a regulatory circuit that encodes the individual’s gene expression in response to the stimuli. A comparison of the max-fitness expression to each individual’s expression yields a fitness score.

(Fig. b) The TM samples the probability distribution of possible binding configurations of regulator genes X, Y, and Z binding to the upstream DNA of gene A. Only three configurations are shown in this cartoon, but typical analysis would use 10,000+ samples. Configurations are only shown for binding at gene A, but in practice, configurations are computed for all genes in each genome. The specificity matrix of each bound regulator contributes a number of bound bits of DNA; bits are translated to expression levels using a variant of the Hill Equation. The expression levels from all samples are then weighted averaged into a single expression value.

(Fig. c) The TM computes gene expression for a single timeslice. By using the output of the TM as the input for the next timeslice, the TM is used iteratively to compute gene expression over multiple timepoints in a simulated cell’s lifespan.

(Fig. d) The simulated population can evolve a solution to the regulatory problem over multiple generations by cycling through the steps of a genetic algorithm: compute expression and fitness for each individual, reproduce the population selectively, and then mutate the population.
