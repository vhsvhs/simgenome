SimGenome

Simulated directed evolution of gene regulatory circuits

Written by Victor Hanson-Smith :: EMAIL victor.hanson-smith@ucsf.edu

Overview | Download and Install | Tutorial | Command-Line Reference

Usage 1:

$> simgen --outdir [path] --psampath [path] --urspath [path] --rulepath [path] [Optional Parameters]

Usage 2:

$> simgen --outdir [path] --poppath [path] --rulepath [path] [Optional Parameters]

--outdir [path], where [path] is a directory into which all output from this simulation will be written. SimGenome will overwrite any previous contents in this folder, and therefore it is strongly suggested that this folder not be shared with other data or previous analaysis.

--rulepath [path], where [path] is the filepath to a text file listing the fixed gene expression inputs and their corresponding regulatory problems.

--psampath [path], where [path] is the filepath to a text file describing the position-specific affinity matrices (i.e., PSAMs) for all transcription factors in one genome. Every individual in the population will be initialized with these PSAMs.

--urspath [path], where [path] is the filepath to a text file containing the upstream regulatory sequences (i.e. URSs) for all genes in one genome. Every individual in the population will be initizlied with these URSs.

--poppath [path], where [path] is the filepath to a saved population written by SimGenome. If this option is present, it will override values for --psampath and --urspath. The option --poppath is often combined with the option --startgen N, where N is not zero, in order to continue a previously saved simulation at a later generation.

Optional Parameters

--verbosity N, where N can range from 0 to 100. Higher values of N produce more output on the screen, while lower values produce less output. Values of 30 and higher will produce very verbose output files that will reduce the performance of SimGenome, although this output can be useful for comprehensively analyzing your simulated conditions. The default value is N = 3.

--runclean will explicitly remove any existing files in the output directory before the simulation begins. This is useful if you are re-running a previous analysis and want to prevent stale output from polluting your results.

--cooppath [path], where [path] is the filepath to a text file containing cofactor interaction definitions. If this file is not specified, then all co-factor interactions will be initialized to 1.0, indicating no specific interaction. See the examples for more information about formatting this file.

--startgen N will start the simulation at generation N.

--maxgen N will stop the simulation when it reaches generation N.

--popsize N will create a population of size N. Each individual will be initialized as a clone of the URSs, PSAMs, and co-factor interactions described in --urspath, --psampath, and --cooppath (if specified). This parameter is ignored if you also specify --poppath.

--maxgd N, where N is the maximum distance, in nucleotide sites, over which two co-factors can interact. Co-factor interactions are strongest when the co-factors are nearby, and the interaction degrades over distance. The default value for this parameter is 3 sites.

--niid N will set the number of independent and identically distributed (I.I.D.) samples to draw from the distribution of binding occupancies. Higher values will increase the accuracy of SimGenome, but will also take longer to compute. The default value is 5000.

Mutation Parameters:

--nomu will disable the mutation step in the genetic algorithm.

--urs_mu X will set the mutation rate for URSs to X, where X is nucleotide substitutions per sequence site.

--psam_mu X will set the mutation rate for PSAMs to X, where X is the probability of mutating a single site in a PSAM.

--urslenmu X, where X is the probability of inserting or deleting sites into each URS.

--urslenmulen X, where X is the mean number of nucleotide sites inserted or deleted with each URS length mutation.

--psamlenmu X, where X is the probability of mutating the length of each PSAM. The mutation can be an insertion or a deletion of a PSAM site.

--psamlenmulen X, where X is the mean number of PSAM sites inserted or deleted with each PSAM mutation.

--coopmu X, where X is the probability of increasing/decreasing each co-factor interaction.

--coopmusize X, where X is the mean multiplier by which each co-factor mutation will increase or decrease the interaction.
